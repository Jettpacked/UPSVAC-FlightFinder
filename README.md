# UPSVAC Route Finder

**UPSVAC Route Finder** is a simple Windows application that lets you find flight routes between airports using UPSVAC data.

---

## Features

- Select your **Aircraft**, **Departure Airport**, and **Arrival Airport** from searchable lists
- Find:
  - **Fewest Legs Route** (least number of stops)
  - **Least Distance Route** (shortest total distance)
- Automatically loads and uses live route data from UPSVAC
---

## How to Run

### Using the Executable

1. Open the `FlightFinder.exe` application.
2. Wait while the application grabs the latest available routes
3. In the Application:
   - Click **Select Aircraft** to choose your aircraft.
   - Click **Select Departure Airport** to choose your starting airport.
   - Click **Select Arrival Airport** to choose your destination airport.
   - Click **Find Routes** to calculate your routes.
4. Results appear in the output box at the bottom and will let you know if there are no routes available between the two airports with the selected aircraft.

---

### Notes

- The app requires an **active internet connection** to load routes.
- IT WILL NOT WORK IF YOU DO NOT HAVE INTERNET CONNECTION
- Only airports and aircraft available in UPSVAC will appear.
